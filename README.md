## 📍 프로젝트 소개
**센스있는 신입사원**이 되고 싶다면? **신입사UP**!

현대 직장에서 성공하려면 단순한 기술력 이상의 무언가가 필요합니다.
바로 **비즈니스 매너**와 업무 효율성을 겸비한 **균형 잡힌 역량**입니다.
신입사UP을 통해 신입사원들에게 필요한 모든 것을 제공받아보세요.

**비즈니스 매너**부터 **실질적인 업무 스킬**, **최신 트렌드** 학습까지!
신입사UP, 당신의 직장생활을 업그레이드합니다.
